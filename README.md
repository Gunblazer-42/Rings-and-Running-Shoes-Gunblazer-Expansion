# A Rings & Running Shoes Expansion
A FoundryVTT version of the [expansion by Gunblazer42](https://docs.google.com/document/d/1KrbbNfF8Hz-42IcXJ4UurXd5CG-HQRtMU-w0k748rJk/edit?tab=t.0) for Rings &amp; Running Shoes, a Powered by the Apocalypse-based, Sonic the Hedgehog-themed tabletop role playing game. 
# What's Included
* A colleciton of Moves to expand upon existing Playbooks
* The Masked, a Playbook themed around magical girls, superheroes, and tokusatsu characters, focused on having two identities, one potentially a secret identity.
